02 — The Studio

One umbrella,
many small things.

Atsumi Labs LLC is an independent software studio. We design, build, and operate small consumer products end to end — the kind of focused, unglamorous tools we wanted to exist and couldn’t find done well.

The name comes from the Japanese atsumi (集み) — a gathering. That’s the whole idea: a single, trustworthy roof for a growing collection of apps, so each one inherits the same standards for privacy, craft, and longevity.

  • Privacy first No selling data. We collect the minimum and say so plainly.
  • Built to last Small, dependable software we intend to keep running.
  • Craft over hype Fewer features, finished properly.
